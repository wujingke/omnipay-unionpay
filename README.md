Labs7IN0\UnionPay
===

[![Build Status](https://travis-ci.org/7IN0-labs/UnionPay.svg?branch=master)](https://travis-ci.org/7IN0-labs/UnionPay)

An UnionPay library with psr-4 autoloader.

## Features

Support for:

 * UnionPay WEB
 * UnionPay WAP
 * UnionPay Non-redirect
 * UnionPay Back pay
